# AI Self-Healing Test Automation Framework

## Overview

An AI-powered test automation framework that automatically analyzes test failures, identifies root causes, and provides healing recommendations.

## Features

* Automated Test Execution
* Failure Categorization
* Root Cause Analysis
* Healing Recommendations
* CSV Report Generation
* Extensible Agent-Based Architecture
